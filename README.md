# A World of Warcraft addon integreting multiple tools made by HBLyx
## **Downloads**:  https://www.curseforge.com/wow/addons/hblyx-tools

## **Features**
### **Focus Interrupt**: support Focus Interrupt task in various ways, and helps users to identify whether the cast of focus is interruptible/interrupt ready/ interrupt in cooldown.
### **焦点打断**: 以多种方式支持焦点打断，并帮助用户识别当前焦点施放的技能是否可打断/已准备好打断/处于打断冷却状态

#### Standalone version 单独版本: [https://www.curseforge.com/wow/addons/focusinteruptsound-midnight](https://www.curseforge.com/wow/addons/focusinteruptsound-midnight)

### **Combat Indicator**: display a text reminder when enter/leave combat and automatically disappear after a specified fade time with optional sound alert
### **战斗指示器**: 当进入/脱离战斗时显示一个文本提示并且播放一个可选的语音警报

### **Combat Timer**: display a MM:SS combat timer which records the combat duration with many customize options with it.
### **战斗时间**: 显示并记录战斗时间(MM:SS)以及多个自定义选项

### **BattleRes** Indicator: display the BattleRes states(works with retail but not Beta so far)
### **战复指示器**：显示战复状态（目前适用于正式服，但不适用于测试服）。

### **M+ Panel Enhance**: Display score and dungeon name and clickable M+ dungeon portal on the M+ Panel
### **大秘境面板增强**: 在M+面板上显示分数、地下城名称以及可点击的M+地下城传送门

### **Warlock Reminders**: Icon+Text reminder for pet and healthstones for warlocks.
### **术士提醒**: 为术士提供宠物和治疗石提醒

### To customize the addon, please use: ESC->Options->Addons->HBLyx\_Tools
### 若需要自定义,请使用: ESC->选项->插件->HBLyx\_Tools
*   Under the Test mode, you can drag to adjust position of all elements shown (recommendation: after adjustment, please turn off the test mode and reload)
*   在测试模式下，您可以拖动调整所有显示元素的位位置(建议：调整完成后，请关闭测试模式并重新加载)
